About The Project

This project is a homework of the course IS601: Web System Development at New Jersey Institute of Technology. In this homework, I did research on virtualization, containerization and the advantages and the disadvantages of these two methods. I have also added 3 tutorials on how to create a web server on your computer, about some basic linux commands and how to set up Docker on Pycharm to write Python codes.

Contents of this webpage:

  • What is virtualization?
  • What is containerization?
  • Virtualization vs Containerization
  • A tutorial on how to setup a web server on your local computer using docker
  • A list of basic Linux commands and what they do as well as how to log into your docker container.
  • A tutorial about how to connect Docker with Pycharm to write python code.
