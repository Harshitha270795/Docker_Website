How to use Docker and Pycharm to write python code?

Prerequisites

    Make sure these prerequisites are met:
  • Docker is installed, as described on the page Docker Docs. You can install Docker on the various platforms.
  • You have a stable Internet connection.
  • Before you start working with , make sure that the Docker plugin is enabled. The plugin is bundled with PyCharm and is activated by default. If the plugin is not activated, enable it on the Plugins page of the Settings/Preferences dialog

Configure Docker in Pycharm

To configure Docker in Pycharm you can do like this: Go to the setting of Pycharm (or preference for Mac). Go to Build, Execution, Deployment and choose Docker

Run Docker

To run Docker, you can simply open the Dockerfile and simply right click in the Dockerfile and choose Run

To configure the Run process, you can look to the top-right corner, and click on the Add Configuration ... button

The Configuration page is follow. You can change the container name, the image name in this page. Remember to check the option Run built image

Run Tab

When the process of running, Pycharm will create the Run tab for run process. You can see 2 special tabs in this Run tab: Build log and Attached Console. The first one logs the information about the building images and containers process and the second on shows you the results of running your python codes.
